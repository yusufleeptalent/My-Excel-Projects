# 📊 Excel Retail & Sales Analysis Project
*Data Technician Bootcamp – Week 1 Project*

## 📌 Project Overview
This project was completed as part of a **Data Technician bootcamp**, focusing on developing strong foundational skills in **Microsoft Excel** for data cleaning, transformation, analysis, and visualisation.  
Using retail and sales datasets, the project demonstrates proficiency with formulas, PivotTables, filtering/sorting, categorisation logic, and charts.

---

## 🧰 Skills & Techniques Used

### ✔️ Excel Formulas  
Applied a variety of formulas to analyse and enrich datasets:

- `SUM()` – Total commission and sales values  
- `SUMIF()` – Conditional totals  
- `AVERAGE()` / `AVERAGEIF()` – Average commission calculations  
- `DATE()`, `MONTH()`, `YEAR()` – Extracting time-based insights  
- `UNIQUE()` – Identifying distinct entries  
- `VLOOKUP()` – Lookup and referencing  
- `SWITCH()` – Categorising products based on sales volume (High / Medium / Low)

---

### 🔍 Data Cleaning & Preparation
- Used **filters** to sort demographic and sales data (e.g., Age → largest to smallest)  
- Cleaned and formatted imported datasets  
- Applied **sorting** to structure data for analysis  
- Ensured fields were ready for use in PivotTables and formulas  

---

### 📈 PivotTables & Analysis
Created multiple PivotTables to understand:

- Sales volume segmented by **County** and **Product**  
- Customer demographics by **Age Group**, **Country**, and **Gender**  
- High‑performing product categories and markets  

These tables provided meaningful insights into sales patterns and customer behaviour.

---

### 📊 Charts & Visualisations
Developed charts to visually communicate findings:

- PivotCharts for bike sales and customer trends  
- Bar/column charts for product comparisons  
- Visual dashboards demonstrating sales distribution  

---

## 🗂️ Project Tasks Completed
- Filtering and sorting retail‑sales data  
- Calculating commission totals and averages  
- Creating PivotTables for multi‑dimensional insights  
- Categorising products using the `SWITCH()` function  
- Designing charts for effective visual storytelling  
- Exploring retail and bike sales datasets to surface insights  

---

## 📁 Included Outputs
- Retail dataset analysis  
- PivotTables summarising sales by county, product, and demographics  
- SWITCH‑based categorisation column  
- Charts and visual outputs from the visualisation labs  

---

## 🎓 Key Learning Outcomes
This project strengthened skills in:

- Data manipulation and cleaning  
- Building and customising PivotTables and PivotCharts  
- Applying intermediate/advanced Excel formulas  
- Creating categorisation logic within datasets  
- Using visualisations to support data storytelling
